ANIMATIONS BREAKDOWN
================================================================================

menuButtonGlow (2s loop)
   Pulsing box-shadow effect on MENU button
   Creates breathing effect similar to core reactor
   Attracts user attention to available actions

menuDropdown (0.4s)
   Smooth menu expansion with cubic-bezier bounce
   Starts at -10px Y offset and 80% scale
   Ends at full visibility with 1.0 scale
   Effect: scaleY(0.8) → scaleY(1) with easing

overlayFadeIn (0.5s)
   Fade-in effect for update overlay
   Backdrop blur gradually appears
   Smooth transition from 0 to 5px blur
   Professional appearance for critical operations

Hover Effects (0.3s)
   Background ::before pseudo-element slides left
   Box-shadow intensifies for depth
   Text color brightens on menu items
   Transform: translateY(-2px) for button lift
