================================================================================
                        INTERACTION FLOW
================================================================================

START: User sees interface
  │
  ├─ MENU button has continuous green glow
  │     (attracts attention)
  │
  └─ User hovers over MENU
       │
       └─ Button lifts up 2px
          Background fill slides in from left
          Green glow intensifies
          (feedback: this is clickable)

  └─ User clicks MENU button
       │
       ├─ MENU becomes ACTIVE (enhanced glow)
       │
       └─ Dropdown menu slides down (0.4s)
            With elastic bounce effect
            "Update System" appears in green
            "Configure Keys" appears in cyan
            (feedback: menu is open and ready)

  └─ User hovers over menu item
       │
       └─ Text slides left 5px
          Cyan/Green glow appears
          Background gradient fills left-to-right
          (feedback: this item is selectable)

  └─ User clicks menu item
       │
       ├─ Menu closes automatically
       │
       ├─ MENU button returns to normal glow
       │
       └─ Selected action executes
            Update overlay appears (fades in)
            Keys setup dialog opens
            (feedback: action completed)
